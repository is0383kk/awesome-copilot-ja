# 🎯 再利用可能なプロンプト

特定の開発シナリオやタスク向けの使い切りではないプロンプトテンプレート。モード、モデル、利用可能なツールを指定してプロンプト文面を定義します。
### 再利用可能なプロンプトの使い方

**インストール方法:**
- 使用したいプロンプトの「VS Code」または「VS Code Insiders」インストールボタンをクリックします
- `*.prompt.md` ファイルをダウンロードし、プロンプトコレクションに手動で追加します

**実行方法:**
- インストール後、VS Code のチャットで `/prompt-name` を使用します
- コマンドパレットから `Chat: Run Prompt` コマンドを実行します
- VS Code でプロンプトファイルを開いた状態で実行ボタンを押します
