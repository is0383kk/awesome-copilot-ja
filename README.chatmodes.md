# 💭 カスタムチャットモード

カスタムチャットモードは GitHub Copilot Chat の振る舞いとツールを特定用途向けに定義し、タスクやワークフローに合わせた高精度のコンテキスト対応支援を可能にします。
### カスタムチャットモードの使い方

**インストール方法:**
- 使用したいチャットモードの「VS Code」または「VS Code Insiders」インストールボタンをクリックします
- `*.chatmode.md` ファイルをダウンロードし、コマンドパレットから VS Code に手動でインストールします

**有効化/使用方法:**
- チャットモードの設定を VS Code の設定にインポートします
- VS Code のチャットインターフェイスからインストール済みのチャットモードにアクセスします
- VS Code Chat の選択肢から目的のチャットモードを選択します
